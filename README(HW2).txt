F74056166 方嘉祥 資料結構 HW2 project 

程式的運作方式為:
	生成8*8棋盤
	生成knight能走的8種路線->以array儲存
	清空棋盤
	每次棋盤大小為n，n值為1~8
	for(隨棋盤大小變化的n){
		選定初始位置
		for(n*n的步數){
			step1:	生成以knight位置為準，knight下一步可能走的八種路線，並刪除違反規則的路線
			step2:	以step1的結果，再進行下一步的預測，刪除違反規則的路線，並計算可行路線的數量
			step3: 	現在我們有了[如果knight走了這條路，那他下一步能走的路線數量]的值
			step4:  選定step3的最小值路線為knight的下一步
			step5:	knight移動，並將第幾步和位置記錄在棋盤上
			如果棋盤尚未走完且無法再移動，將這n*n棋盤判定為無解
		}
		if(knight成功走完棋盤){
			印出棋盤
		}
		else{
			印出no solution
		}
	}
